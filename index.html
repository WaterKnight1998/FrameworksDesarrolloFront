Bootstrap es un kit de herramientas de código abierto para desarrollar con HTML, CSS y JS. Realice rápidamente prototipos de sus ideas o cree toda su aplicación con nuestras variables y mixins Sass, sistema de cuadrícula sensible, componentes precompilados extensos y complementos potentes creados en jQuery.

Sass es un procesador CSS.

Un preprocesador CSS es una herramienta que nos permite generar, de manera automática, hojas de estilo, añadiéndoles características que no tiene CSS, y que son propias de los lenguajes de programación, como pueden ser variables, funciones, selectores anidados, herencia, etcétera.

Estas características de los procesadores nos permiten, además, que el CSS que se genera sea más fácil de mantener y más reutilizable.

LESS es un preprocesador de texto, como una ampliación de las hojas de estilos, que se comporta como un lenguaje de programación. Nos permite el uso de variables, funciones y operaciones aritméticas entre otras cosas.

Hemos de tener claro que LESS no reemplaza a CSS, es más, el resultado de la compilación de LESS es un fichero CSS totalmente funcional.

Variables El nombre de la variable se debe preceder de: “$”.br*

Si sobreescribes una variable "global" localmente, la variable global toma el valor local. 

                      $font-stack: Helvetica, sans-serif;
                      $primary-color: #333;
                      
                      body {
                          font: 100% $font-stack;
                          color: $primary-color;
                      }
El nombre de la variable a diferencia de Sass se precede de: “@”. 

                      @nice-blue: #5B83AD;
                      @light-blue: @nice-blue + #111;
                      
                      #header {
                          color: @light-blue;
                      }

Inicialmente presentado por Google en octubre de 2010, AngularJS ha sido una opción viable para muchos desarrolladores de todo el mundo. Angular es un marco con conceptos básicos estructurales, y permite a los desarrolladores de aplicaciones utilizar HTML como idioma principal. Desde Angular 2, ha ofrecido a los desarrolladores poderosas opciones para reducir el tamaño de la aplicación. Su capacidad de eliminación de código acompañada de dependencia de enlace de datos provoca una reducción importante de tamaño.

Con la ayuda de la interfaz de línea de comandos de Angular, es una opción factible para los desarrolladores en 2020. Permite configurar los módulos, así como la implementación de la zona. Teniendo en cuenta su naturaleza adaptable y diminuta, hay pocas dudas de que los desarrolladores preferirían AngularJS como lo han hecho en los últimos 10 años.

Vue.JS es un Framework basado en modelo-vista-vista-modelo. Creado en el año 2014 por el propio Evan You de Google, Vue JS es una de las principales soluciones de Framework en el mercado hoy en día. Permite a los desarrolladores crear interfaces de usuario junto con complejas aplicaciones de una sola página. A pesar de que es nuevo en comparación con otros idiomas en la biblioteca JS, Vue es extremadamente popular entre los desarrolladores.

Uno de los principales elementos de Vue.js es su naturaleza adaptable con respecto a otros idiomas. Los procesos de configuración simples junto con otros complementos lo convierten en un compañero viable para los desarrolladores. Es por eso que Vue.js está preparado para triunfar en el mundo de la programación en 2020.

React es una biblioteca para construir el modelo front-end de una aplicación mediante JavaScript. En otras palabras, es una biblioteca de JavaScript para desarrollar interfaces de usuario en aplicaciones. Facebook lo introdujo en los primeros días como React-Native, desde entonces las comunidades de desarrolladores y Facebook lo han gestionado ellos mismos.

Desde su introducción en 2018, React ha dominado la escena del desarrollo de aplicaciones. Una de las principales ofertas de react incluye interfaces de usuario interactivas y atractivas, que permiten a los desarrolladores ahorrar horas de trabajo. React dio grandes saltos en 2019, y ahora está listo para dominar 2020 también. Destaca el modelo de suspenso y el modo concurrente, que permite a los desarrolladores lograr aplicaciones futuristas de alta nivel.

Flutter es el kit de herramientas de interfaz de usuario de Google para crear hermosas aplicaciones compiladas de forma nativa para dispositivos móviles, web y de escritorio desde una única base de código, flutter usa el lenguaje de Dart. Todo dentro de una aplicación Flutter es un widget en Flutter, desde un simple Texto, Botones o cualquier diseños de pantalla. Estos widgets se organizan en un orden jerárquico para mostrarse en la pantalla. Los widgets que pueden contener widgets dentro se llaman Container Widget. La mayoría de los widgets de diseño son widgets de contenedor, excepto los widgets que hacen un trabajo mínimo como "Text Widget". Existen dos tipos Al escribir una aplicación, normalmente creará nuevos widgets que son subclases:

  • Stateless: Un widget sin estado es facil de entender ya que es un widget que no cambia por ejemplo un simple Button o un Text.
  • Stateful: Un widget con estado es dinámico: por ejemplo, puede cambiar su apariencia en respuesta a eventos desencadenados por las interacciones del usuario o cuando recibe datos. Checkbox , Radio , Slider , InkWell , Form y TextField son ejemplos de widgets con estado. Subclase de widgets con estado StatefulWidget .
